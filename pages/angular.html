Angular JS

Angular é uma plataforma de aplicações web de código-fonte aberto e front-end baseado em TypeScript liderado pela Equipe Angular do Google e por uma comunidade de indivíduos e corporações. Angular é uma reescrita completa do AngularJS, feito pela mesma equipe que o construiu.

Fonte: MDN Web Docs

Angular JS

Angular foi anunciado no ng-Europe conference 22-23 de setembro de 2014. As mudanças drásticas na versão 2.0 criou uma considerável controvérsia entre os desenvolvedores.

Em 30 de abril de 2015, os desenvolvedores Angular anunciaram que o Angular 2 foi transformado de Alfa a Developer Preview. O Angular 2 mudou-se para o Beta em dezembro de 2015, e a primeira versão foi publicada em Maio de 2016. A versão final foi lançada em 14 de setembro de 2016.

O Angular 17 foi lançado em 06 de novembro de 2023:

  • Nova documentação e uma melhor experiência do usuário para todos os níveis de desenvolvedor.
  • Novo construtor de aplicativos com ESBuild e Vite.
  • Novo visual, mesma estrutura.

O que posso fazer com Angular?

  • São aplicativos web que carregam uma única página HTML e atualizam dinamicamente o conteúdo à medida que o usuário interage com a aplicação.
  • Oferecem uma experiência de usuário mais fluida e rápida, similar à de aplicativos desktop.
  • São aplicativos web que se comportam como aplicativos nativos, oferecendo recursos como instalação na tela inicial, notificações push e acesso offline.
